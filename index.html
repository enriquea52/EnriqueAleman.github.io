    Robot Exploration Using a Hybrid Approach: Frontier + Sample Based Exploration

    This project involved designing an exploration approach for indoor 2D mapping on a Tutlebot 2. The path planning algorithm used is a custom made RRT* that takes into account the differential constraints of the bicycle model to plan smooth and short paths for the robot to explore.

      D* Lite + Voronoi Diagram for Safe, Fast and Efficient Motion Planning and Replanning

      Project developed in collaboration with Vsevolod Hulchuk. This project shows the implementation of the original D* Lite algorithm in Python for path planning and replanning on a graph which is defined by a Voronoi decomposition of the environmnet. The current implementation defines the environment with point obstacles.

        • Learn more

        Pose Based EKF SLAM Using LiDAR and ICP for Scan Matching

        EKF SLAM implemnetation using robot poses to grow the state vector instead of external features. The prediction uses the robot's odometry, while the update step uses the ICP scan matching algorithm to find the translation from two overallping LiDAR scans and filter the map.

        Collision Risk Assessment Using an Intel Realsense D435i RGB-D Camera for Obstacle Avoidance

        Implementation of a 3D perception algorithm to find the Time To Collision (TTC) from the robot's trayectory to any of the points from the pointcloud measured using the RGB-D camera, allowing to compute the risk to collide, visualize it and create a control signal for the robot to automatically evade the closest obstacle.

          Mobile Manipulation Using a Differential Drive Robot and a 4-DoF Palletizing Robotic Arm for Mobile Pick and Place Applications via Task Priority Control

          Mobile Pick and Place system using the Task Priority Control algorithm for redundant mechanisms. The object to be picked and placed is detected using ArUco markers, these markers also allow to compute the 3D position of the object in the global coordinate system, for the robotic system to pick it using a vacuum (end-effector).
